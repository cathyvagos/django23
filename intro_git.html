GIT

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Como verificar se temos instalado: git --version

Para criar um repositório local: git init

Ver como status: git status


Comando para adicionar ficheiro para o staging area (área de preparação): git add 'filename'

git add nomeficheiro1 nome ficheiro2

Comando para adicionar todos os ficheiros da pasta para o staging area / área de preparação: git add .


Para gravar um snapshot / uma versão do nosso código: git commit -m "descrição"

Para efetuar o commit é necessário ter definido a informação do autor / configuração do git:

    git config --global user.name "nome utilizador"
    git config --global user.email emailutilizador

Para visualizar os vários commit/snapshots efetuados:

    Todo o log : git log
    Ou também para ver uma só linha git log --oneline

Ramificação do código - criar branch e efetuar o merge

Para criar uma nova ramificação e indicar o seu nome: git branch nomebranch

Para indicar o ramo/branch onde estamos: git branch

Para mudar de ramo/branch: git switch 'outroramo' ou git checkout 'outroramo'

Para eliminar um ramo/branch: git branch -d 'nomeramo'

Para fazer um merge de um ramo/branch com o master: git merge 'nomeramo'
